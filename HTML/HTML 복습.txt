HTML5 누워서 휴대폰으로 보기용

기본=========================================

<!DOCTYPE html> : HTML5 문서임을 명시
<html> : HTML 문서의 루트요소 정의
<head> : 문서의 metadata 정의 (title,style,meta,link,script,base 등)
<title> : 툴바에 표시, 즐겨찾기 제목, 검색엔진 결과페이지의 제목
<body> : 브라우저를 통해 보이는 내용(content)
<h1>~<h6> : 제목
<p> : 단락

<태그이름 속성이름="속성값">
속성명은 소문자
속성값은 따옴표 ( " , ' ) 로 감싸는 습관 가지기

제목,단락======================================

<h> 태그는 제목 외 검색엔진들의 키워드가 된다.
※ big, bold 등의 태그를 사용하지 않도록 !

<h> <p> 태그는 위 아래로는 약간의 margin 삽입됨

<br> : 줄바꿈
<pre> : 작성내용 그대로(개행,띄어쓰기) 적용
- <pre>태그 내에 작성된 텍스트의 글꼴(font)은
- 고정폭 글꼴(fixed-width font)로 자동변환

<hr> : 수평 가로 구분선

서식=========================================
(중요) : 중요하다는 의미도 포함 -> 검색엔진이 더 중요하게 인식
<b> : 굵게
<strong> : 굵게+중요

<i> : 기울임꼴
<em> : 기울임꼴+중요

<mark> : 하이라이팅 효과 (형광펜)

<del> : 삭제표시 (중간 가로줄)

<ins> : 밑줄효과

<sup>,<sub> : 위,아래 첨자 (지수 표시나 화학식처럼)
