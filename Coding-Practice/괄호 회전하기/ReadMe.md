처음에는 각 괄호마다 열릴때 +1, 닫힐때 -1 을 계산하는 방식으로 모두 0일 시 정답 처리를 했다.   
그러나 input 값이
```
[{(})]
```
처럼 들어온다면, 올바른 괄호가 아닌데도 정답 처리를 해버려서 좋은 방법을 생각하다 스택을 생각했다.
   
스택에 괄호를 쌓아두고, 닫히는 괄호가 있을때, 가장 최근 괄호가 올바른 열린 괄호일 경우 pop 을 해주었다.
