DFS/BFS 너무 어렵다..   
개념을 알고있어도 실제로 적용할 땐 어떤 식으로 짜야할지 막막하다.   
계속 하다보면 되겠지 아무생각없이 그냥 하자.   
   
메인메서드에서 2번 호출한 이유는 입력값으로 들어오는 배열의 첫 인자도 부호가 변경되기 때문이다.   
DFS 로 설명하자면 최상위 노드가 1, -1 인 2개의 그래프가 있고,   
각 그래프를 탐색하기 위해 2번 사용해줬다.
