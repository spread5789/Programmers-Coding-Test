중요한 것은 데이터의 형태이다.   
x의 범위는 -천만 ~ +천만 까지이다.
   
만약 x 는 10,000,000   
n 은 1000   
이 주어졌다고 해보자.   
   
그렇다면 answer 의 마지막 값은 int 형의 범위를 벗어나는 100 억이 될 것이다.   
   
문제의 제한 조건에 따라, 형변환을 반드시 해 주어야 한다.
