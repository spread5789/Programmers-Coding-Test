회고
==
연속된 문자들만 비교하면 됐는데 각각 비교해서 시간을 한참 날려먹었다..   
예를 들어보자면, abcdabcdefghabcd 라는 문자가 있을때,      
2abcdefghabcd 가 되어야할텐데,
3abcdefgh 로 만들어버렸다.. 문제를 잘 읽고 뻘짓을 하지 않도록 해야겠다.
