문제에 이미 접근방식.. 그러니까 어떻게 풀어야하는지 제시해준다.   
   
그러니까.. Pseudo-Code 가 이미 문제에 주어지니, 그대로 구현하기만 하면 됐었다.   
처음에 구현해봤는데 바로 통과하길래 살짝 당황하긴 했다.
