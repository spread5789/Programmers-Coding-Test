==
첫번째 (1의자리가 아닌) 숫자부터 나눠서 추출해야할까...   
String 변환 후 각 자리를 split, 배열에 담아서 더해줄까...   
이 방법은 자원도 많이 먹는 코드라서 정말 쉬운 문제인데도 꽤 고민했던 문제이다.   
   
역으로, 1의자리부터 추출을 해볼까...   
10으로 나눈 나머지가 1의 자리인데.. 하다가   
   
그렇다면 10으로 계속 나눈 값을 10으로 나눈 나머지가 각 자리수가 될 수 있구나 생각했다.
