==
카카오톡의 오픈채팅 닉네임 관련 문제이다.   
   
map은 같은 키의 값을 넣어주면 값이 갱신되는 특징이 있으므로, uid만 보면서 최종닉네임을 갱신시켰다.   
모두 갱신시킨 후에는 입장,퇴장 문구만 보면서 uid(키)에 해당하는 닉네임(값)만 꺼내왔다.
