==
먼저, 문제를 봤을때 가장 많은 부서에게 물품을 사주려면   
신청 예산이 적은 부서 먼저 사주는 것이 더 많은 부서들의 물품을 사줄 수 있을 것이라는 생각이 들었다.   
그 다음부터는 간단하다.   
   
각 부서들이 신청한 예산을 오름차순으로 정렬 후, 차례대로 사주고 예산이 부족하다면 안사주면 되는 것이다.
